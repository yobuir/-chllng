Clone the project 

```bash
git clone https://github.com/yobuir/-chllng.git
```
```bash
cd -chllng
```


Install program dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Run the test:

```bash
npm run test
```

Run the coverage
```bash
npm run coverage
```
